# ARM Datapath Mini Project

Ce dépôt contient la conception et la simulation d’un datapath ARM simplifié en VHDL.  
Projet académique réalisé dans le cadre du cours d’architecture des ordinateurs à l’UQTR (Automne 2025).

## Objectif
Implémenter un sous-ensemble du processeur ARM incluant :
- Unité arithmétique et logique (ALU)
- Fichier de registres
- Décodeur d’instructions
- Vérification de conditions (CondCheck)
- Compteur ordinal (PC)
- Mémoire d’instructions (IMEM)
- Registres de flags

## Structure du dépôt
- `src/` : fichiers VHDL
- `tb/` : testbenches
- `doc/` : schémas, diagrammes et annexes
- `.gitignore` : fichiers exclus
- `README.md` : documentation du projet
